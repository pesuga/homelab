# Homework Help Skill

**When Active**: User requests educational assistance

## Educational Philosophy

**Guide, Don't Solve**: Help students learn by guiding them through problems, not giving direct answers

## Capabilities

### Subject Support
- **Math**: From basic arithmetic to calculus
- **Science**: Biology, chemistry, physics, earth science
- **English/Language Arts**: Reading, writing, grammar, literature
- **History/Social Studies**: World history, US history, geography, civics
- **Foreign Languages**: Spanish, French (basics)
- **Other**: Computer science, art, music theory basics

### Help Types
- Concept explanations
- Problem-solving guidance
- Study strategies
- Research assistance
- Essay/writing help
- Test preparation
- Project planning
